# 小波轉換：給傅立葉轉換裝上時間之眼

## TL;DR

小波轉換（Wavelet Transform）解決了傅立葉轉換最大的缺陷：**傅立葉只告訴你「有哪些頻率」，但不告訴你「哪個頻率在什麼時間出現」**。小波改用一個「可伸縮、可平移的母波」當尺，低頻用大尺、高頻用小尺，在每個頻率尺度上都用最合理的解析度。結果是一張二維的時間 × 尺度熱力圖，兼顧時間定位和頻率辨識。廣泛應用於音訊分析、影像壓縮（JPEG 2000）、地震偵測、心電圖、神經科學與深度學習。

---

## 傅立葉的致命缺陷

假設你有兩段音樂：

- **A**：先彈 C 音一秒，再彈 G 音一秒
- **B**：先彈 G 音一秒，再彈 C 音一秒

傅立葉轉換對這兩段的結果**完全一樣**——它只統計「總共有哪些頻率」，不管順序。這意味著它做的是全局統計，丟失了所有時間位置信息。

凡是和「某個頻率在什麼時刻出現」有關的分析——音樂節奏、語音辨識、地震前兆、心電圖異常——傅立葉轉換都力不從心。

### 打補丁：短時傅立葉轉換（STFT）

工程師的第一個修法是把信號切成小段，對每段做傅立葉——即**短時傅立葉轉換（STFT, Short-Time Fourier Transform）**。但它有一個無法克服的硬傷：

- 窗口**大** → 頻率清楚，時間模糊
- 窗口**小** → 時間精準，頻率模糊

更糟的是，窗口大小**固定**，低頻和高頻用同一把尺——這非常不合理。

## 核心譬喻：自動換比例尺的地圖

你在看地圖：

- 找咖啡廳 → 用 1:5000 街道圖：細節清楚，但只涵蓋幾條街
- 看交通幹道 → 用 1:500000 全島圖：覆蓋廣，但街道消失了

小波轉換就像一套**自動換比例尺的地圖系統**：

- 分析**高頻細節**（快速變化）→ 縮放母波使其變小 → 時間定位精準
- 分析**低頻輪廓**（緩慢趨勢）→ 拉伸母波使其變大 → 頻率辨識清楚

### 比喻在哪裡失準

地圖可以同時印出兩個比例尺版本，讓你**同時**對照。但小波轉換仍受**海森堡不確定性原理**制約——在某個尺度上，時間精確度和頻率精確度仍不能同時達到最高，你得到的是「在每個尺度上最合理的分配」，而非「無限精確的兩者兼得」。

## 機制拆解

### 第一步：選一個母波（mother wavelet）

母波（mother wavelet）$\psi(t)$ 是一個「有限長度的小振動」——在某個地方振動，然後衰減消失（這就是「小」波的意思）。常見的母波：

| 母波 | 形狀特徵 | 適合應用 |
|---|---|---|
| Haar | 方波，最簡單 | 教學、邊緣偵測 |
| Morlet / Gabor | 正弦波乘高斯包絡 | 音訊、神經科學 |
| Daubechies | 非對稱，緊支撐 | 影像壓縮、一般工程 |
| Mexican Hat | 高斯二階導數 | 邊緣偵測、地球物理 |

母波必須滿足兩個條件：**積分為零**（上下振動平衡，無直流分量）以及**能量有限**（不無限延伸）。

### 第二步：縮放（scale）與平移（translation）

把母波做兩種操作：

- **縮放**（$a$）：壓縮母波 → 高頻；拉伸母波 → 低頻
- **平移**（$b$）：沿時間軸滑動，問「這個位置有沒有和母波相似的結構？」

連續小波轉換（CWT）的公式：

$$W(a, b) = \frac{1}{\sqrt{a}} \int_{-\infty}^{\infty} f(t)\, \psi\!\left(\frac{t - b}{a}\right) dt$$

其中 $a > 0$ 是縮放參數，$b$ 是平移參數，$\frac{1}{\sqrt{a}}$ 是能量歸一化係數。

### 第三步：輸出是二維熱力圖（scalogram）

結果是一張**時間 × 尺度**的熱力圖——哪個時刻、哪個頻率尺度有強烈響應，顏色越深表示越相似。這讓瞬間發生的事件（如地震 P 波抵達）可以被精確定位在時間軸上。

## 小波 vs. 傅立葉：核心對照

| | 傅立葉轉換 | 小波轉換 |
|---|---|---|
| 基礎函數 | 無限延伸的正弦波 | 有限長度的母波 |
| 時間信息 | 完全丟失 | 保留 |
| 頻率解析度 | 均一固定 | 低頻精、高頻粗（自適應） |
| 輸出維度 | 一維（頻率） | 二維（時間 × 尺度） |
| 適合分析 | 穩態、週期信號 | 非穩態、瞬變信號 |
| 可逆性 | 完全可逆 | 完全可逆（滿足容許條件） |

## 跨領域串聯

### 時間錯位：JPEG → JPEG 2000

早期 JPEG 用離散餘弦轉換（DCT，傅立葉變體）分塊壓縮。壓縮率過高時出現明顯**方塊感**——因為每個小塊獨立處理，邊界不連續。

JPEG 2000 改用小波轉換，一次處理整張圖。壓縮劣化變成整體模糊，視覺上自然許多。這是「換了數學工具，改變了失真的物理樣貌」的典型案例。

連結方式：**時間錯位**——影像壓縮走傅立葉 → 小波的路，就像音訊領域更早走過的。

### 同構：大腦視覺皮層的內建小波

大腦 V1 視覺皮層有一批神經元，對「特定位置 × 特定方向 × 特定空間頻率」的邊緣敏感——這個結構和 Gabor 小波（Morlet 的變體）的數學形式**幾乎完全一致**。

視覺神經科學界認為，大腦的初級視覺處理可能內建了一套小波分析機制。這也是 CNN（卷積神經網路）早期卷積層學到的濾波器，往往長得像 Gabor 小波的原因。

連結方式：**同構**——工程數學和生物神經機制共享同一套信息處理邏輯。

## 延伸追問

1. **離散小波轉換（DWT）** 如何用「高通/低通濾波器組」實現多尺度分解？為什麼它比連續小波轉換快得多，計算複雜度是多少？
2. 不同的母波（Haar、Morlet、Daubechies）各有什麼特性？工程師選擇母波的直覺是什麼？
3. 小波在**地震波分析**裡怎麼幫助偵測前兆信號？P 波和 S 波的時頻特徵有何不同？
4. **CNN 的卷積濾波器**和小波基函數的親緣關係：可學習的濾波器是否在做一種廣義的小波分解？
5. **壓縮感知（Compressed Sensing）** 如何利用信號在小波域的稀疏性，用遠少於奈奎斯特採樣率的數據重建完整信號？（這是 MRI 加速掃描的關鍵）
